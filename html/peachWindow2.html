How to upload your queries

  1. Write and save your queries on a simple (non-enriched) text file.
  2. Recommendations:
    - Follow these
    naming conventions.    ⚬ Field names should use camelCase.
       ⚬ Type names should use PascalCase.
       ⚬ Enum names should use PascalCase.
       ⚬ Enum values should use ALL_CAPS.
       ⚬ Consider globally unique names for your variables.
  3. Drag & Drop the file on the peach-colored area on the right.
  4. Click Save Button
  5. Locate the Directory "RELAY"
  6. Replace the existing imported.js file
  7. Close this window
